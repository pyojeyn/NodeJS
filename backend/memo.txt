JWT(JSON Web Token)
- 데이터가 JSON으로 이루어져 있는 토큰
- 두 개체가 서로 안전하게 데이터를 주고 받을 수 있도록 하는 기술

세션기반
- 사용자가 로그인 → 세션id를 발급 → id를 브라우저 쿠키에 저장 → 사용자가 요청할 때 마다 세션을 
조회하고 로그인 여부를 확인하고 처리
- 단점 : 서버가 여러대라면 세션을 공유해야하는 방법이 필요

토큰기반
- 로그인 이후 서버가 만들어주는 문자열
- 사용자 로그인 정보 + 서버에서 발급한 서명
